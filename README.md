## Base Application for Laravel and Vue.

## Environment

* PHP + Laravel + Inertia.js + Vue.js + TypeScript + Wayfinder + Tailwind.css + SQLite

* Composer 2.9.x
* PHP 8.4.x
* Laravel 13.2.x

* Node 25.8.x
* NPM 11.11.x
* Inertia 3.0.x
* Vue 3.5.x
* Wayfinder 0.1.x
* Tailwind CSS 4.1.x

* SQLite 3.37.x


## Installation

composer run setup


## Start Application

composer run dev
